Tags Para Marcação de Títulos

As Tags de h1 a h6 são utilizadas para marcar títulos na página web. Elas trabalham de uma forma que pode parecer um pouco estranho para algumas pessoas, pois tem uma relação inversa onde os números maiores representam os títulos menores e não o contrário. Fica mais fácil de interpretar se ao invés de relacionar a tamanho relarcionarmos a ordem de importância. Onde o primeiro título mais importante em uma página é representado pelo h1, o segundo h2 e assim segue até o h6. É claro que os títulos mais importantes são maiores e tem maior destaque.

É importante também ressaltar a importância da utilização correta das tags para que o site tenha a sua busca otimizada pelos motores de buscas

Tags Para Criação de Listas

As Tags de listas são utilizadas para trazer uma melhor organização ao conteúdo. Utilizada para criação de índices, tópicos etc.

Existem dois tipos bastante utilizados que são a lista ordenada "ol" e a lista desordenada "ul". Seguem exemplos:

Exemplo de lista ordenada

  1. Arroz
  2. Feijão
  3. Carne

Exemplo de lista desordenada

  • Arroz
  • Feijão
  • Carne

Mais Tags

Existem tags para marcar diversos tipos de conteúdo nas páginas. Algumas delas carregam formatações específicas junto e se aprofundando no estudo do HTML com o complemento do CSS é possível criar sites com muitos recursos.

Seguem mais alguns exemplos de Tags:

Tag "a"

A Tag "a" representa uma âncora pois o seu conteúdo está ancorado em outro lugar. Utilizado para direcionar o usuário para outras páginas por meio de links ou para a mesma página por meio de um ID de um elemento HTML. Exemplo:

O Melhor do Mundo

Tag "Small"

A tag "Small" é mais um exemplo de tags de formatação, pois deixa o conteúdo menor.

Texto sem "Small" X Texto com a Tag "Small"

Tag "hr"

Já a tag "hr" insere uma linha horizontal cortando a página. H = Horizontal R = Row que significa linha em inglês.

Tag "Sub"

A tag sub é utilizada para formatar elementos para que fiquem menor do que o texto ao seu lado e posicionados próximo a linha inferior.

Rsub

Tag "Sup"

A tag sup é utilizada para formatar elementos para que fiquem menor do que o texto ao seu lado e posicionados próximo a linha superior.

Rsup

Tag "Blockquote"

A tag "Blockquote" é utilizadas para marcar citações, ela destaca o texto adicionando um recuo em relação ao texto superior.

Texto com blockquote.
