Tags Para Marcação de Títulos

As Tags de h1 a h6 são utilizadas para marcar títulos na página web. Elas trabalham de uma forma que pode parecer um pouco estranho para algumas pessoas, pois tem uma relação inversa onde os números maiores representam os títulos menores e não o contrário. Fica mais fácil de interpretar se ao invés de relacionar a tamanho relarcionarmos a ordem de importância. Onde o primeiro título mais importante em uma página é representado pelo h1, o segundo h2 e assim segue até o h6. É claro que os títulos mais importantes são maiores e tem maior destaque.

É importante também ressaltar a importância da utilização correta das tags para que o site tenha a sua busca otimizada pelos motores de buscas
